# Instrucciones de uso del programa:

- Paso 1: Abrir la terminal dentro del proyecto de ICE.
- Paso 2: Rehacer la construccion (build) de la compilacion de las clases mediante el comando *.\gradlew build*
- Paso 3: Ejecutar *Server.jar* mediante el comando *java -jar /direccion_de_clase*
- Paso 4: Ejecutar *Client.jar* con el mismo comando anterior
- Paso 5: Ejecutar *Worker.jar* con el mismo comando anterior
- Paso 6: Poner a prueba el programa como se quiera usar...
